# phone-select-mid-2026

Небольшой Hugo-сайт для сравнения смартфонов по личным критериям.

## Где что лежит

- `data/criteria.yaml` - критерии, веса, описания и проверки.
- `data/models/*.yaml` - модели телефонов, оценки по критериям, цены и риски.
- `data/specRows.yaml` - строки таблицы характеристик.
- `data/sources.yaml` - источники, сгруппированные по разделам.
- `content/` - страницы сайта.
- `layouts/` - шаблоны Hugo.
- `static/css/report.css` - стили.
- `input/` - исходные XLSX/заметки, из которых собирались данные.

## Как запустить

Нужен Hugo.

```powershell
hugo server
```

Сайт будет доступен по адресу, который выведет Hugo, обычно `http://localhost:1313/`.

## Как собрать

```powershell
hugo
```

Сборка попадет в `public/`. Эта папка не хранится в git, потому что полностью пересоздается из исходников.

## Как менять рейтинг

Рейтинг на главной считается в шаблоне из веса критерия и оценки модели:

```text
итог = сумма(criteria.weight * model.criteria[criterion_id].score)
максимум = сумма(criteria.weight)
```

Чтобы изменить результат, правь веса в `data/criteria.yaml` или оценки/заметки моделей в `data/models/*.yaml`.
